###  What does Dapr do for MCP servers?

Using Dapr, developers can interact securely with MCP servers, enable fine-grained ACLs between for MCP clients and get built-in tracing and metrics, as well as resiliency policies to handle situations where an MCP server might be down or unresponsive.
